# Скрини демо-кейсу

Покладіть сюди реальні скриншоти слайдів презентації / сторінок бізнес-плану
демонстраційного кейсу (майстерня комплектуючих для БпЛА).

Рекомендовано:
- формат WebP або AVIF (з JPG-fallback), 2x для ретини;
- імена, напр. `case-1.webp`, `case-2.webp`, `case-3.webp`;
- після додавання замініть заглушки у `src/components/CaseTeaser.astro`
  та на сторінці `src/pages/case-bpla.astro` на `<img>` зі `srcset`.
