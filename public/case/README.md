# Слайди кейсів

Слайди згенеровані у фірмовому стилі на основі презентацій та фінмоделей
реальних кейсів (HTML → PNG, 1280×800 @2x).

- `case-1-cover.png`, `case-2-koshtorys.png`, `case-3-finmodel.png` —
  кейс «Майстерня БпЛА» (сторінка `/case-bpla`).
- `salon-1-cover.png`, `salon-2-koshtorys.png`, `salon-3-finmodel.png` —
  кейс «Салон краси повного циклу» (сторінка `/case-salon-krasy`).

Список кейсів для тизера на головній — `src/data/cases.ts`.

Замінити на реальні скрини презентації/плану можна, зберігши ті самі
імена файлів (співвідношення 1280×800). Фото юриста та фінансистки —
окремо у `src/components/Author.astro` і `FinancePartner.astro`.
