# Слайди кейсу

case-1-cover.png, case-2-koshtorys.png, case-3-finmodel.png — слайди
демо-кейсу (майстерня БпЛА), згенеровані у фірмовому стилі на основі
презентації та фінмоделі. Використовуються в hero, блоці кейсу на
головній та на сторінці /case-bpla.

Замінити на реальні скрини презентації/плану можна, зберігши ті самі
імена файлів (співвідношення 1280×800). Фото юриста для блоку «Хто
готує ваш план» додається окремо у src/components/Author.astro.
